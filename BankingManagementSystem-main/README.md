# Banking Management System

A Java Console Application developed using Object-Oriented Programming concepts.

## Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Balance Check
- Money Transfer
- View All Accounts
- PIN Verification

## Technologies Used

- Java
- OOP Concepts
- ArrayList
- Scanner Class

## Concepts Implemented

- Classes and Objects
- Constructor
- Encapsulation Basics
- Loops
- Conditional Statements
- Menu Driven Programming

## Project Structure


BankManagementSystem/
│
├── BankAccount.java
└── BankManagementSystem.java


## Sample Menu


1. Create Account
2. Deposit Money
3. Withdraw Money
4. Check Balance
5. View All Accounts
6. Transfer Money
